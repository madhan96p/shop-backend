            Define the catalog with categories and product details.        
        cart.py: 
            Handle cart management features (add/remove items).        
        payment.py: 
            Simulate payment options (UPI, PayPal, etc.).        
        admin.py: 
            Implement admin functions like adding/removing products and categories.
        main.py: 
            Write the main driver code to initialize the app and handle user input.
