Create a new repository (e.g., shopping-app-backend).

shopping-app-backend/
├── admin.py             # Admin functionality
├── cart.py              # Cart management functionality
├── catalog.py           # Catalog management (products, categories)
├── login.py             # User login functionality
├── payment.py           # Payment processing
├── main.py              # Main program to manage user/admin interface
├── utils/               # Utility functions
│   └── db_mock.py       # Mock data for users, products, etc. (Optional)
├── README.md            # Project documentation
└── venv/                # Virtual environment directory (generated by virtualenv)

# Set Up Virtual Environment 
- pip install virtualenv
- virtualenv venv
- venv\Scripts\activate
- pip install flask
